计算广告学

广告的基础知识

  • 广告（advertising）的定义：

    《现代广告学》：广告是由已确定的出资人通过各种媒介进行的有关产品的，通常是有偿的，有组织的，综合的，劝服的非人员的信息传播活动

  • 广告的主体：广告主，媒介，受众

  • 广告的本质：借助媒介，完成较低成本的用户接触

  • 品牌广告：品牌形象，提高长期转化率

  • 效果广告：短期用户转换

  • 广告有效性模型：

    • 选择：曝光（广告位），关注（兴趣，不要打断用户任务等）
    • 解释：理解（匹配），信息接受（认同，认可）
    • 态度：保持（艺术性），购买（价格敏感，杀熟）

  • 传统广告策略，在线广告创意原则

  • 广告与销售的区别：潜在用户，明确需求

  • 在线广告的独特性

    • 技术与计算导向
    • 可测量性
    • 标准化

  • 在线广告市场

  • 在线广告的核心计算问题：max (a),to ROI(u,a,c),find the best match between a given context c,and a suitable ad a.

    • 优化角度：
      • 特征提取，特征工程，机器学习，从众定向
      • 微观优化：CTR预测
      • 宏观优化：竞价市场机制
      • 受限优化：在线分配
      • 强化学习：探索利用
      • 个性化重定向：推荐技术
    • 从系统角度
      • 候选查询：实时索引
      • 特征存储：No-SQL技术
      • 离线学习：hadoop
      • 在线学习：流计算
      • 交易市场：实时竞价

  • 在线广告计算的主要挑战

    • 大规模，高并发
    • 动态性
    • 丰富的查询信息
    • 探索与发现

  • 搜索（相关性），广告（投资回报率）与推荐（用户兴趣）的区别

    • CPM:固定eCPM

    • CPA，CPS,ROI市场：动态CTR，点击价值

    • 受众定向平台：海量数据挖掘平台，机器学习平台，分布式架构
    • 高并发投送系统
